# cafe-map

## 프로젝트 소개
- Kakao Maps API를 이용하여 주변의 카페를 검색 
- 카페에 대해 한줄평, 평점을 남길 수 있음


## 기간
- 2022.01.02 ~ 추가 중

## 기술 스택
- Backend : **Spring Boot 2.6.1, Java 8**
- Database : **Mysql**
- Tool : **Intellij IDEA, AqueryTool**
- **Restful API**
